Changelog `v1.0.15` <small>(2020-12-21)</small>
=========================================================

SDK
---------------------------------------
### Added
* Added additional encoding type `basic-auth` to the [OAuthParameterDefinition](../docs/oauth/oauth-objects)
* Added new request contentType property to the [OAuthRequestDefinition](../docs/oauth/oauth-objects)
* Added new properties `shipByDate`, `isPremiumProgram`, `premiumProgramName` to the [ShippingPreferencesObject](../docs/reference/shipping-preferences.mdx)
* Added [shippingOptions](../docs/reference/shipping-options.mdx) to the following methods.
  * [Create Shipment](../docs/reference/methods/create-shipment.mdx)
  * [Rate Shipment](../docs/reference/methods/rate-shipment.mdx)
* Added new properties `shipByDate`, `isPremiumProgram`, `premiumProgramName`, `isGift` to the [ShippingPreferencesObject](../docs/reference/shipping-preferences.mdx)
* Added new properties `lastModifiedDateTime`, `originalOrderSource`, `requestedWarehouse` to the [salesOrders](../docs/reference/methods/get-sales-orders-by-date.mdx)
* Added new property `statusMappings` to the [range](../docs/reference/methods/get-sales-orders-by-date.mdx)
* Added new properties `orderNumber`, `details`, `location` to the [Sales Order Item Object](../docs/reference/sales-order-item.mdx)
* Added new property `details` to the [Shipment Cancelled Object](../docs/reference/methods/shipment-cancelled.mdx)
* Added new properties `details`, `notifyBuyer`, `fulfillmentCost`, `insuranceCost`, `notes`, `currency` to the [Shipment Created Object](../docs/reference/methods/shipment-created.mdx)
* Added new property `details` to the [Shipment New Package Object](../docs/reference/new-package-object.mdx)
