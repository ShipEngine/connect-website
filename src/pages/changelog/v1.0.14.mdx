Changelog `v1.0.14` <small>(2020-11-16)</small>
=========================================================

SDK
---------------------------------------
### Added
* Added optional providerId to Order and Carrier Application
* Added optional tokenExpirationLengthTimeUnit to the [OAuthTokenProperties](../docs/v1/oauth/oauth-config)
* Added availableCountries to the [DeliveryService](../docs/v1/reference/delivery-service.mdx)
* Added language to the [Transaction](../docs/v1/reference/transaction.mdx)

### Updated
* Made contents.quantity optional on [New Package Object](../docs/v1/reference/new-package-object.mdx)
* Made contents.unitPrice optional on [New Package Object](../docs/v1/reference/new-package-object.mdx)
* Made contents.totalPrice optional on [New Package Object](../docs/v1/reference/new-package-object.mdx)

### Removed
* Removed originCountries to the [DeliveryService](../docs/v1/reference/delivery-service.mdx)
* Removed destinationCountries to the [DeliveryService](../docs/v1/reference/delivery-service.mdx)
