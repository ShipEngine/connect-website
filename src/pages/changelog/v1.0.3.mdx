# Changelog `v1.0.3` <small>(2020-09-01)</small>

## CLI

- Fixed support for the `expectedErrorMessage` attribute within the [`connect.config.js`](../docs/v1/testing/config).

## SDK

#### [createShipment()](../docs/v1/reference/methods/create-shipment.mdx) Method

- `ShipmentConfirmation.packages` (in the return value) is now optional

#### [rateShipment()](../docs/v1/reference/methods/rate-shipment.mdx) Method

- `Rate.packages` (in the return value) is now optional

#### [trackShipment()](../docs/v1/reference/methods/track-shipment.mdx)

- `TrackingInfo.packages` (in the return value) is now optional
