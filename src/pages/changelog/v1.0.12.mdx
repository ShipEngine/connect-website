Changelog `v1.0.12` <small>(2020-10-14)</small>
=========================================================

SDK
---------------------------------------
### Removed
* Removed unused `InsuranceProvider` enum
* All maximum length for string properties in the SDK have been removed.


CLI
----------------------------
### Added
* Add [connect_all_fields](../docs/v1/carrier-app/tests/connect/connect-all-fields) acceptance tests.
* Add [trackShipment_return](../docs/v1/carrier-app/tests/track-shipment/track-shipment-return) acceptance tests.
* Add [rateShipment_return](../docs/v1/carrier-app/tests/track-shipment/rate-shipment-return) acceptance tests.
* Add [cancelPickups_next_day](../docs/v1/carrier-app/tests/cancel-pickup/cancel-pickups-next-day) acceptance tests.
* Add [cancelPickups_next_day](../docs/v1/carrier-app/tests/cancel-pickup/cancel-pickups-next-day) acceptance tests.
* add [cancelShipments_multiple](../docs/v1/carrier-app/tests/cancel-shipmentkup/cancel-shipments-multiple) acceptance tests.

### Changed
* `cancelShipments` acceptance test was renamed to [cancelShipments_single](../docs/v1/carrier-app/tests/cancel-shipmentkup/cancel-shipments-single).


### Bugs
* Fixed error where `connect test` would fail if the `connect.config.js` file was missing
