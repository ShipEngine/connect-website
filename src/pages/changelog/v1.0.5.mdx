# Changelog `v1.0.5` <small>(2020-09-03)</small>

## CLI

### Added

- Added a new [`connect apps`](https://github.com/ShipEngine/connect-cli#connect-apps) command to view all apps belonging to the given API key
- Added a new [`connect logs`](https://github.com/ShipEngine/connect-cli#connect-logs) command to view deployment logs from the CLI

### Changed

- The [`connect publish`](https://github.com/ShipEngine/connect-cli#connect-publish) command now supports order apps
- The [`connect info`](https://github.com/ShipEngine/connect-cli#connect-info) command's output has been changed to a table layout
- The `connect` root command is now an alias for [connect help](https://github.com/ShipEngine/connect-cli#connect-help-command)

### Fixed

- Silent deployment failure in [`connect publish`](https://github.com/ShipEngine/connect-cli#connect-publish)
