### Added
* Added additional encoding type 'basic-auth' to the [OAuthParameterDefinition](../docs/oauth/oauth-objects)
* Added new request contentType property to the [OAuthRequestDefinition](../docs/oauth/oauth-objects)
* Added new properties 'shipByDate', 'isPremiumProgram', 'premiumProgramName' to the [ShippingPreferencesObject](../docs/reference/shipping-preferences.mdx)
* Added [customShippingOptions](../docs/reference/custom-shipping-options.mdx) to the following methods.
  * [Create Shipment](../docs/reference/methods/create-shipment.mdx)
  * [Create Manifest](../docs/reference/methods/create-manifest.mdx)
  * [Rate Shipment](../docs/reference/methods/rate-shipment.mdx)
  * [Schedule Pickup](../docs/reference/methods/schedule-pickup.mdx)
