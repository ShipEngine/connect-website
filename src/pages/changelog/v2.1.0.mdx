# Changelog `v2.1.0` <small>(2020-09-03)</small>

## Added

#### CLI

- [connect apps](https://github.com/ShipEngine/connect-cli#connect-logs) command adds the ability to view all apps belonging to the given API key
- [connect logs](https://github.com/ShipEngine/connect-cli#connect-logs) command adds the ability to view deployment logs from the CLI

## Changed

#### CLI

- [connect publish](https://github.com/ShipEngine/connect-cli#connect-publish) command now supports order apps
- [connect info](https://github.com/ShipEngine/connect-cli#connect-info) command's output has been changed to a table layout
- `connect` root command is now an alias for [connect help](https://github.com/ShipEngine/connect-cli#connect-help-command)

## Fixed

#### CLI

- Silent deployment failure in [connect publish](https://github.com/ShipEngine/connect-cli#connect-publish)
