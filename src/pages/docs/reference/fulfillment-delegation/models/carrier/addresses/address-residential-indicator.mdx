AddressResidentialIndicator
===============================================
An Enum to represent whether an address is residential or commercial


<Reference>  
  <Field name="Unknown">
    <Description>
      "Unknown" - The residential or commercial status of this address is unknown 
    </Description>
  </Field>
  <Field name="Yes">
    <Description>
      "Yes" - The address is a residential address
    </Description>
  </Field>
  <Field name="No">
    <Description>
      "No" - The address is not a residential address - it is a commercial address
    </Description>
  </Field>
</Reference>
