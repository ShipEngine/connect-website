Billing Line Item Object
===============================================
A billing line item object is used to represent billing line items on shipments.


Properties
--------------------------------

This table lists the properties of a billing line item object and identifies those properties that are required. 


<Reference>

  <Field name="billing_category" required={true}>
    <Type>
      [billing category](#billing-categories)
    </Type>
    <Description>
      The [billing category](#billing-categories) for this line item.
    </Description>
  </Field>

  <Field name="carrier_description" type="string" required={false}>
    <Description>
      If the carrier provides a description from their API about the billing charge (not specific to the user). Maximum length of 100 characters.
    </Description>
  </Field>

  <Field name="carrier_billing_code" type="string" required={false}>
    <Description>
      If the carrier provides a billing code for the billing charge.
    </Description>
  </Field>

  <Field name="memo" type="string" required={false}>
    <Description>
      This will often be empty; however, if there was additionally specific information about the charge it should go here, could also be a reference number. Maximum length of 250 characters.
    </Description>
  </Field>

  <Field name="amount" type="object" required={true}>
    <Description>
      Total amount and currency of the line item.
    </Description>
  </Field>

  <Field name="amount.amount" type="string" required={true}>
    <Description>
      The amount of the line item.
    </Description>
  </Field>

  <Field name="amount.currency" type="string" required={true}>
    <Description>
      The currency of the line item.
    </Description>
  </Field>
</Reference>
