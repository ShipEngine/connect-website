CustomsContentTypes
===============================================
An Enum of the possible values for type of customs content for a declaration


<Reference>  
  <Field name="Merchandise">
    <Description>
      "Merchandise" - The package contains merchandise.
    </Description>
  </Field>
  <Field name="Documents">
    <Description>
      "Documents" - The package contains documents.
    </Description>
  </Field>
  <Field name="Gift">
    <Description>
      "Gift" - The package contains a gift.
    </Description>
  </Field>
  <Field name="Sample">
    <Description>
      "Sample" - The package contains a commercial sample, such as a flooring sample.
    </Description>
  </Field>
  <Field name="ReturnedGoods">
    <Description>
      "ReturnedGoods" - The package is a returned shipment.
    </Description>
  </Field>
</Reference>
