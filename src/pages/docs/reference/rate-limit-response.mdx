RateLimitResponse
===============================================
This represents the error response for when the 3rd party api's are rate limiting us

<Reference>
  <Field name="retry_after_seconds" type="number" nullable={true}>
    <Description>
      The number of seconds one should wait before making this request again
    </Description>
  </Field>
  <Field name="retry_after_time" type="string" nullable={true}>
    <Description>
      The (ISO 8601) datetime (UTC) of when the next request can be made @example "2021-03-31T18:21:14.858Z"
    </Description>
  </Field>
  <Field name="throttling_context" type="any" nullable={false}>
    <Description>
      Information returned by the 3rd party systems that might be useful in logs
    </Description>
  </Field>
</Reference>
