`acknowledgeOrders()`
===========================
This method is called when a sales order is imported into one of our e-commerce applications.


Syntax
-----------------------------------------------
<CodeWrapper>
```javascript
module.exports = async function acknowledgeOrders(transaction, salesOrderNotifications[] {
  // Your code here
}
```

```typescript
import { Transaction, SalesOrderNotification, AcknowledgeSalesOrder} from "@shipengine/connect-sdk";

  transaction: Transaction,
  notifications: SalesOrderNotifications
): Promise<void> {
  // Your code here
}
```
</CodeWrapper>



Parameters
-----------------------------------------------

### `transaction`
A [transaction object](../transaction.mdx) containing information about the transaction and session state.

### `notifications[]`
An array of notifications regarding sales orders that have been imported.

<Reference>

  <Field name="id" type="string" nullable={false}>
    <Description>
      The marketplace's unique ID for the sales order. This string must be between `1` and `100` characters and must not contain newline characters.
    </Description>
  </Field>

  <Field name="identifiers" type="object" nullable={false}>
    <Description>
      Your own identifiers for this sales order.
    </Description>
  </Field>

  <Field name="orderNumber" nullable={true} type="string">
    <Description>
      The customer facing identifier of the sales order.
    </Description>
  </Field>

  <Field name="importedDate" nullable={false}>
    <Type>
      [DateTime](./../date-time.mdx)
    </Type>
    <Description>
      The sales order associated with this shipment.
    </Description>
  </Field>

</Reference>
