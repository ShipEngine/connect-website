Authorization
===============================================
Represents the authorization information sent with every request


<Reference>
  <Field name="access_token" type="string" nullable={true}>
    <Description>
      The access token of the user making the request for this carrier.
    </Description>
  </Field>
  <Field name="connection_context" type="any" nullable={true}>
    <Description>
      Additional source-specific information needed to connect to the API.
    </Description>
  </Field>
</Reference>
