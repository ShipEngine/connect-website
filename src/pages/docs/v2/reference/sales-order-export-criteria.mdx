SalesOrderExportCriteria
===============================================
This represents the requested criteria for exporting order sources

<Reference>
  <Field name="from_date_time" type="string" nullable={true}>
    <Description>
      The (ISO 8601) datetime (UTC) associated with the beginning of the range we would like to pull orders from @example "2021-03-31T18:21:14.858Z"
    </Description>
  </Field>
  <Field name="to_date_time" type="string" nullable={true}>
    <Description>
      The (ISO 8601) datetime (UTC) associated with the end of the range we would like to pull orders from @example "2021-03-31T18:21:14.858Z"
    </Description>
  </Field>
</Reference>
