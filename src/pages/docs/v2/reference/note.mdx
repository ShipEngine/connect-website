Note Object
===============================================
Note objects are used throughout the Order Source specifications to describe order notes.


Properties
--------------------------------

This table lists the properties of a note and identifies those properties that are required. 

<Reference>

  <Field name="type" type="string" required={true} nullable={false}>
    <Type>
      [note type](#note-types)
    </Type>
    <Description>
      The type of note being sent.
    </Description>
  </Field>

  <Field name="text" type="string" required={true} nullable={false}>
    <Description>
      The contents of the note.
    </Description>
  </Field>

</Reference>
