## Demension Details Object

This model represents the dimensions for a package represented in its original unit, inches, and centimeters.

<Reference>
  <Field name="dimensions_in_centimeters" nullable={false}>
    <Type>
      [demension object](./units.mdx#Demension-Object)
    </Type>
    <Description>
      The dimensions in Centimeters rounded to the fourth decimal place.
    </Description>
  </Field>

{' '}

<Field name="dimensions_in_inches" nullable={false}>
  <Type>[demension object](./units.mdx#Demension-Object)</Type>
  <Description>The dimensions in Inches rounded to the fourth decimal place.</Description>
</Field>

{' '}

<Field name="source_dimensions" nullable={false}>
  <Type>[demension object](./units.mdx#Demension-Object)</Type>
  <Description>The dimensions in the original dimensions rounded to the fourth decimal place.</Description>
</Field>

  <Field name="source_dimensions_unit" type="string" nullable={false}>
    <Description>
      The unit of the source dimensions. Either centimeters or inches.
    </Description>
  </Field>
</Reference>
