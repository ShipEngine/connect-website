# Create Label Types

This page lists the valid values for the types seen in a [Create Label](./methods/create-label) request.

## Delivery Confirmation Types

This is the list of valid values for delivery confirmation types.

<Reference>

{' '}

<Field name="None">
  <Description>No confirmation is requested.</Description>
</Field>

{' '}

<Field name="Delivery">
  <Description>Delivery confirmation is requested.</Description>
</Field>

{' '}

<Field name="Signature">
  <Description>
    A signature is required. This signature may be a neighbor, building manager, or the recipient can authorize the
    release of the package (without being present).
  </Description>
</Field>

{' '}

<Field name="AdultSignature">
  <Description>A signature of an adult is required.</Description>
</Field>

{' '}

<Field name="DirectSignature">
  <Description>Only supported by FedEx. The signature of somebody at the address is required.</Description>
</Field>

</Reference>
