Packing Slip Object
===============================================
Packing Slip objects are used throughout the Order Source specifications to describe the requested packing slip from an order source for a fulfillment.

<Reference>

  <Field name="url" type="string" required={false}>
    <Description>
      The url from which the packing slip can be downloaded.
    </Description>
  </Field>

</Reference>
