# Pickup Window Object

An Pickup Window object is used to describe time windows for scheduled pickups.

<Reference>

{' '}

<Field name="time_zone_iana" type="string" nullable={false}>
  <Description>
    The local time zone represented in IANA. See
    https://www.iana.org/time-zones.
  </Description>
</Field>

{' '}

<Field name="pickup_date" type="string" nullable={false}>
  <Description>
    The local date the pickup will occur. Formatted per
    https://tools.ietf.org/html/rfc3339.
  </Description>
</Field>

{' '}

<Field name="start_time" type="string" nullable={true}>
  <Description>
    The time relative to the time_zone_iana that the carrier says that the
    pickup could occur. Formatted per https://tools.ietf.org/html/rfc3339.
  </Description>
</Field>

{' '}

<Field name="end_time" type="string" nullable={true}>
  <Description>
    The end of the window when the pickup could be made relative to the
    time_zone_iana. Formatted per https://tools.ietf.org/html/rfc3339.
  </Description>
</Field>

</Reference>
