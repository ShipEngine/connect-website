# Document Object

A document object is used to represent a document.

## Properties

This table lists the properties of a document and identifies those properties that are required.

<Reference>

{' '}

<Field name="type" required={true}>
  <Type>[document type[]](#document-types)</Type>
  <Description>
    An array of [document types](#document-types) describing this document.
  </Description>
</Field>

{' '}

<Field name="data" type="string" required={true}>
  <Description>Base64 encoded data for the document.</Description>
</Field>

{' '}

<Field name="format" type="string" required={true}>
  <Description>
    The requested document format for this document. Valid values include the
    following: * `PDF` * `ZPL` * `PNG`
  </Description>
</Field>

</Reference>

## Document Types

This table lists the valid values for document types.

<Reference>

{' '}

<Field name="label">
