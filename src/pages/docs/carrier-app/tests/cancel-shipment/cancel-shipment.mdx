Carrier App Tests
==================================

### `cancelShipment`
This test exercises your app's ability to successfully cancel a shipment.


Test Parameters
-------------------------
This test has many parameters [that can be customized](#customizing-parameters) in your [`connect.config.js` file](./../../../testing/config.mdx).
Any parameters you don't override will use their default values.
