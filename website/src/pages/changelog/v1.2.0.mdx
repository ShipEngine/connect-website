Changelog `v1.2.0` <small>(2021-03-12)</small>
=========================================================

CLI
---------------------------------------
### Deprecated
* Combined with the Connect package

Connect
---------------------------------------
### Updated
* Now contains all of the code from the CLI package
* New command to generate infra files for ShipEngine internal deployments
* New projects will now import the SDK directly

### Deprecated
* Now emits a deprecation warning when imported into an app

SDK
---------------------------------------
### Updated
* Added 'pickupLocations' properties to Addresses for Order Apps
* Added 'documents' array to the shipping preferences for Order Apps
